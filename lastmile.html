LASTMILE

LASTMILE smart mailboxes provide a seamless user experience for receiving, collecting, and sending mail and packages using advanced technologies.

Project Details

  • Cyber Security: Robust protection for user data and communications.
  • Management and Control: Full control over mailbox settings via a dedicated app.
  • Mobile Notifications: Receive notifications and alerts on your mobile device about mail and package arrivals.

My Role in Testing LASTMILE

As a Software QA Engineer, I was responsible for conducting System Testing and Integration System Testing to ensure that LASTMILE's smart mailbox system functions flawlessly across all its components and user interactions.

System Testing

In my role, I conducted thorough system testing to ensure that the entire LASTMILE platform worked seamlessly, including:

  • End-to-End Testing: I validated the complete user journey, from receiving notifications to opening mailboxes and managing settings via the mobile app.
  • Security Testing: I ensured that cyber security features worked as expected, protecting users' sensitive information from unauthorized access.
  • Usability Testing: I evaluated the app's user interface and experience, ensuring that it provided intuitive, easy-to-use controls for residents.

Integration System Testing

I also performed integration testing to validate the communication between the smart mailbox hardware and software components. This involved:

  • API Integration: Testing the API interactions between the mobile app and the mailbox system to ensure accurate data transfer for notifications, package details, and control commands.
  • System Compatibility: Verifying that the app successfully interacted with mailbox control systems, ensuring smooth functionality for opening compartments, managing user settings, and reporting issues.

Through comprehensive system and integration testing, I ensured the LASTMILE smart mailbox system delivered a reliable, user-friendly, and secure experience for residents, enhancing the overall functionality and performance of the platform.
